Array:

An array is an ordered collection of data (either primitive or object depending upon the language). Arrays are used to store multiple values in a single variable. This is compared to a variable that can store only one value.

Filter Method:

  • This method create a new array filled with elements that pass a test provided by a function.
  • This method does not execute the function for empty elements.
  • This method does not change the oringial array.

Example: 
                                var number = [25,30,15,55,40,10];
                                number.filter(funtion(number)
                                {
                                    return number>=30;
                                });
                                 

                                // output: [30,55,40]

Flat Method:

Flat() method to convert an original array into a new array. It does this by collecting and concatenating the sub-arrarys in an array into a single array.

Example: 
                            const days = ["mon","tues",["wed","thurs",["fri","sat"]],"sun"];
                            console.log(days.flat());
                             
                            
                            //output: ["mon","tue","wed","thrus","fri","sat","sun"]

Spread Operator Method:

spread operator (. . . . .) allows us to quickly copy all or part of an existing array or object into another array or object.

Example: 
                            var indoor = ["Chess","carrom","Dice"];
                            console.log(...indoor);
                            var outdoor =["Cricket","Football","Basketball"];
                            var game = [...indoor,...outdoor];
                            console.log(game);

                            //outpuy:['Chess','Carrom','Dice']
                            //output:['Chess','carrom','Dice','Cricket','Football','Basketball']
